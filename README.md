# movie-rec-app
# Movie Recommendation App

A web-based movie recommendation platform integrating with The Movie Database (TMDb) API.


## Architecture

**Backend**: Node.js + Express ('backend/server.js') — serves both the API and static frontend files on port 3000
**Frontend**: Vanilla JavaScript, HTML5, CSS3, Bootstrap 5 (CDN)
**External API**: TMDb API (key hardcoded in 'backend/server.js')
**Storage**: 'localStorage' for saved movies and user preferences (no database)

## Admin + CRUD

Admin panel is at https;//localhost:3000/admin - needs to be logged in as admin.
Adding a movie - fill in title, genre, year and press add movie.
Editing a movie - click edit button to change the form and save.
Deleting a movie - click delete on the movie to remove it.
The list loads automatically when the page opens + a refresh button to reload manually.


## Database Schema

Movies are stored in MongoDB in the 'userLoginSystem' database in the movies collection.

Each movie has:
'_id'      # auto generated by mongoDB,
'title'    # name of movie,
'genre'    # type of movie,
'year'     # year of release


## Client Pages & Features

### Home
- **Discover Genres** - 15 genre buttons that can be clicked to load 20 movies of selected genre via TMDb discover.
- **Upcoming Releases** - 5-slide Bootstrap carousel pulling movies releasing in the next 4 months, arrow buttons or dot indicators can be clicked to skip between slides.

### Movie Detail
  - Any movie on any page can be cicked to view; movie poster, title, genre, release date, description, cast, rating, reviews'
- **Save Movie** - button which when clicked saves a full snapshot (id, title, poster, rating, releaseYear, releaseDate, genreIds, genres) to 'localStorage', can be clicked again to unsave.
- **Rate & Review** - click the preferred amount of stars to rate movie and/or type a review in the designated box and click 'Submit' button to leave a review.

### Trending
- **Popular This Week** - grid of the week's most popular movies pulled live from TMDb.

### For You
- **Your Saves** - full list of all saved movies.
- **Based On Genre** - 6 movie recommendations matching the most common genre from saved movies.
- **Based On Release Year** - 6 movie recommendations matching the most common year from saved movies.

### Settings
  - Side navbar with hash-based tab switching (`#account`, `#accessibility`, `#preferences`, `#faqs`).
- **Account** — view and update your profile details.
- **Accessibility** — adjust the app's text and/or icon size (XS to XL) using the size buttons; add colour filter/overlay to entire site and/or change language using respective drop-down boxes; toggle dark mode, auto-scroll and/or text-to-speech on or off using switches.
- **Preferences** — update genre and language preferences using drop-down boxes.
  - Above settings can be saved by clicking 'Save Changes' situated under the settings on each page.
- **FAQs** — common questions and answers.
- **Sign Out** - click to be taken back to the 'Register/Log In' page.
- **Delete Account** - click to permanently delete account.
