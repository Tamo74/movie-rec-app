What Does This App Do?

This website helps people find movies they will enjoy!

- Browse thousands of movies
- Read movie descriptions and ratings
- Write your own reviews
- Get personalized movie recommendations
- Admins can add and manage movies


 Technology We Used

| Part | Technology:
-  Frontend (what you see) | HTML, CSS, JavaScript 
-  Backend (the brain) | Node.js, Express 
- Database (storage) | PostgreSQL 
- Movie Data | The Movie Database (TMDb) API
